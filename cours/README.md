## Comment utiliser ce livret

**Option A — Google Colab (recommandée, sans installation)**
1. Ouvre le notebook du module dans `notebooks/`
2. Clique sur le badge « Ouvrir dans Colab » ou importe le `.ipynb` dans [colab.research.google.com](https://colab.research.google.com)
3. Exécute les cellules dans l'ordre
4. Complète les exercices dans le notebook, puis dans `exercices/ex_XX.py`

**Option B — VS Code en local**
1. Clone le repo : `git clone https://github.com/stefm78/drone-rescue-python`
2. Ouvre le dossier dans VS Code
3. Lis la fiche `.md` du module dans `cours/`
4. Complète l'exercice dans `exercices/ex_XX.py`

> ⚠️ Toujours essayer l'exercice avant de regarder la correction.
